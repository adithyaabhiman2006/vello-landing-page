Vello

Personal Finance Management Application

A cross-platform Flutter app that helps Sri Lankan users track expenses, scan bills with OCR, plan event budgets, and get weekly AI-powered financial guidance — in English, Sinhala, and Tamil.

GitHub Repository Watch Demo

The Problem

Why Sri Lankan users need a finance app designed for their context

Many users struggle to manage personal finances because they forget small daily expenses, overspend without noticing, and find it difficult to plan budgets for bills, events, and festivals.

Existing finance apps do not fully match the Sri Lankan context — especially for language support, cash-based transactions, SMS-based payments, and culturally relevant financial guidance.

Overspending
Forgotten Expenses
Language Barriers
No SMS/Bill Detection

Proposed Solution

Vello bridges the gap with a practical, accessible, and intelligent finance tool

Vello provides a simple personal finance management system with income and expense tracking, category-based budgeting, colour-coded visual dashboards, OCR bill scanning, Gmail/SMS bill detection, weekly AI-based budget recommendations, event & festival budget planning, and savings goal tracking.

The app supports English, Sinhala, and Tamil to improve accessibility for all Sri Lankan users — students, working professionals, and families alike.

System Architecture

A clean, layered design built for scalability and security

UI Layer

  • Flutter Cross-Platform App
  • Dashboard & Charts
  • Bill Scanner & AI Chat
  • Tri-Language Interface

Functionality Layer

  • OCR Handler (Google ML Kit)
  • Gmail / SMS Import Service
  • Rule-Based AI Agent
  • Notification & Sync Manager

Data Layer

  • Firebase Firestore (Cloud DB)
  • SQFLite Encrypted (Local DB)
  • Firebase Cloud Storage
  • Flutter Secure Storage
